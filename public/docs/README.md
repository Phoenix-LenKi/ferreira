# 🚀 Ferreira Odontologia — Projeto Phoenix Repositório

Repositório Oficial contendo do Front-End Premium e da arquitetura do site final da **Dra. Fernanda Ferreira**, desenvolvido usando as diretrizes "Phoenix-Odonto-Orchestrator".

## 📚 Documentação (Pasta `/docs`)
Este projeto contém arquivos comerciais isolados na pasta `/docs` que devem ser lidos e apresentados fora do ambiente Node.js.
  
- `argumento-comercial.html` -> Pitch Deck para os Closers
- `antes-depois.html` -> Transformações listadas (Value Drop)
- `qa-checklist.html` -> Controle de Qualidade de Entrega
- `relatorio-tecnico.html` -> Árvore estrutural, Tokens, e Security Headers.

## 🛠 Como Rodar o Site

Este aplicativo foi construído usando Next.js 14 App Router.

```bash
# 1. Instalar dependências Next / Tailwind
npm install

# 2. Iniciar o servidor de desenvolvimento
npm run dev

# 3. Empacotar para Vercel Deploy (Production Build)
npm run build
```
